# OurTrip

#### 프로젝트 소개
- 항공권 숙박, 티켓, 투어, 액티비티 등을 예매할 수 있는 '마이리얼트립'을 모티브로 한 여행 예약 플랫폼 사이트 '아워트립' 제작
- 개발 기간 : 2022/06/07 ~ 2022/06/16 (10일)
- 시연 영상 : https://www.youtube.com/watch?v=n8d-xT8DhpA
